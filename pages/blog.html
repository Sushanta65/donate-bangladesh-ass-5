Frequently Asked Question(FAQ)

What is the Document Object Model (DOM)?

The DOM (Documnet Object Model) is a programming interface for web development or web documents. Its represent the structure of the HTML document and allows developer or programmer to manipulate/handle the document structure, styles and content also. We can change HTML elements using DOM.

How do you select an element from the DOM?

We know JavaScript provides several methods to select DOM(Document Object Model) elements. Here are some of method:

  • getElementsByTagName()
  • getElementById()
  • getElementsByClassName()
  • querySelector()
  • querySelectorAll()

getElementsByTagName: is using for get all of the element with the tag name.

getElementById: is use for select a specific one element by provided id.

getElementsByClassName: is use for select all of the element which has the same class name.

querySelector: is use for select first child element of an element that matches with css selector.

querySelectorAll: is use for select all elements that matches with css selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a method in JavaScript where we can handle events from parent element. Its means when use one listener for multiple elements, and any other child element if added dynamically on the parent element then the listener will also work for new child. Thats called event delegation.

Event delegation is very usefull method in javaScript. Because its improve performance, simplified event handling, dynamic event adding etc. And thats why event delegation is very usefull.

How do you manipulate an element's attributes and styles using the DOM?

First we have to select the element by usign document method getElementsByTagName or getElementsByClassName or getElementById or querySelector or querySelectorAll. Then set attributes using setAttribute and to change values and its style properties directly. Then we can manipulate the element's attributes and styles using DOM.
